普通查询

条件查询

limit – 返回设定数量的记录

offset – 在查询过程中，跳过指定数量的记录，一般和limit一起使用

除此之外，最经常用到就是where了

  • where相当于程序语言中的if
    • and：相当于 &&
    • or：相当于 ||

= 和 in 的区别？

  • 后接参数数量
    • = 1个
    • in 多个
  • in可以后接返回多个结果的一个子查询

分组 — 可以对重复出现的字段根据他们进行分组

排序

UNION

用于连接两个以上的select语句的结果组合到一个结果集合中，默认会删除结果中的重复元素

连接查询

**等值连接—（内连接）**

读取两张表中符合的数据，即两张表的交集

左连接

读取左边数据表符合的全部数据，即便右边表无对应数据

右连接

读取右边数据表的全部符合的数据，即使左表无对应数据
